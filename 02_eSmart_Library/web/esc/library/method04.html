사이트 이용안내

자녀의 학습/시험 내역을 관리할 수 있습니다.

자녀의 성적을 확인할 수 있습니다.

  • 로그인

    Main 우측 상단의 ‘로그인’을 클릭한 후 가입한 ID로 로그인을 하면, 학부모용 관리 페이지로 이동합니다.

  • 자녀등록

    자녀의 ID와 이름을 입력 한 후 ‘검색’ 버튼을 클릭하면, 자녀의 정보가 표시됩니다.

    자녀의 정보를 확인 한 후 ‘등록’ 버튼을 클릭하며, 자녀 등록이 완료됩니다.

    자녀 등록이 완료되면 자녀학습내역에서 자녀의 학습 / 시험 / 성적표 내역을 확인할 수 있습니다.

  • 자녀학습내역

    ‘자녀’와 ‘교재’를 선택하시면 학습 목록을 확인할 수 있습니다.

    학습과 관련한 성적표는 제공되지 않으며, 학습 횟수만 표시됩니다.

    ‘성적표’ 버튼을 클릭하면 해당 시험의 성적표를 확인할 수 있습니다.

    ‘정/오답 확인’ 버튼을 클릭하면 해당 시험의 정답/오답을 확인 할 수 있습니다.
