구성품 확인하기
슈퍼브이를 받으시면 구성품을 먼저 확인해주세요.
터치펜은 케이스 뒷면 홀더에 있습니다.
학습이 끝나면 구성품을 체크해서 반납해주세요.
(체험교재 미반납)
  • 슈퍼브이 구성품 확인
※ 구성품이 분실 또는 훼손될 경우, 유상으로 비용이 청구될 수 있으므로 유의해주세요.
부팅하기(전원켜기)
상단 전원 버튼을 약 5초간 길게 꾹 눌러 학습기를 켜 주세요.
  • 태블릿 재부팅하기
태블릿 화면 이상 또는 오작동을 할 경우 전원버튼과 볼륨 마이너스 버튼을 동시에 8초 이상 누르면 재부팅이 됩니다.
와이파이 연결
학습할 장소에서 와이파이를 연결해 주세요.
5G 연결 후 속도가 느리거나 끊기면 2G 연결을 권장 드립니다.
  • 연결하기 > Wi-Fi 선택 > 비밀번호 입력
업데이트
새로운 슈퍼브이가 있다면 업데이트를 진행해요.
로그인
학습할 아이디로 로그인 해주세요.
  • 체험회원
아이디 : sv휴대폰번호 (예시: sv01012345678)
비밀번호 : 휴대폰번호 뒷 4자리 (예시 : 5678)
  • 정회원
회원가입 시 입력한 아이디, 비밀번호
회원가입
체험신청 회원은 간편가입을 해주세요.
  • 학습에 사용할 아이디와 비밀번호 생성
학습하기

학습계획
일일학습의 학습계획 주기를 수정할 수 있어요.
  • 메인 > ① 상단 학습계획 버튼
학부모님 휴대폰 인증 후 계획을 수정해요.
  • ② 시간표 관리하기 > ③ 날짜 선택 >
    ④ 휴대폰 번호 인증 > ⑤ 강좌 [수정] >
    ⑥ 학습요일 선택 후 수정 버튼
슈퍼브이 설정
편리한 설정 기능도 이용 해보세요.
  • 메인 > 우측 설정 메뉴
