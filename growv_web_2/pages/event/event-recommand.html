2월 친구추천 이벤트

친구에게 슈퍼리딩 추천하면 친구도 나도 상품권 100% 증정

2월

참여기간
2월 1일(목) ~ 2월 29일(목)
경품발송
24년 4월 17일(수) LMS 개별발송
* 이벤트 대상 기간 내 무료체험 신청 후, 슈퍼리딩 학습을 진행한 회원

무료체험 추천

  • 무료체험 추천한 나에게 5,000원(네이버페이 모바일 금액권) X 추천한 지인수
    • 월 1인당 최대 10만원까지 초대받은 사람이 무료체험 진행 완료 시
      (로그인 기준)
  • 무료체험 추천받은 친구에게 5,000원(네이버페이 모바일 금액권)
    • 초대한 사람이 보낸 링크로 무료체험 신청 후
      진행 완료 시 (로그인 기준)

구매 추천

  • 친구가 구매하면 나에게 30,000(해피머니 문화상품권) X 추천한 지인수
    • 월 1인당 최대 30만원까지
    • 초대받은 사람이 슈퍼리딩 구매 완료 시
    • 초대받은 사람이 24개월 이상 상품을 구매할 경우에 한함
    • 특별상품 제외
  • 추천받고 구매한 친구에게 30,000(해피머니 문화상품권)
    • 초대한 사람이 보낸 링크로 무료체험 진행 후 슈퍼리딩 구매 완료 시
      (로그인 기준)
    • 24개월 이상 상품을 구매할 경우에 한함
    • 특별상품 제외

꼭 확인해주세요!
  • 본 이벤트는 ‘추천받은 분’이 기존 슈퍼리딩 무료체험 이력이 없는 ‘신규회원’일 경우에만 참여가 인정됩니다.
  • 반드시 친구에게 공유받은 ‘고유 추천링크’를 통해 무료체험 신청을 해야 참여가 인정됩니다.
  • 슈퍼리딩 무료체험 추천 이벤트의 경우, 추천받은 사람이 무료체험을 진행하지 않은 경우(태블릿 로그인), 혜택적용에서 제외됩니다.
  • 슈퍼리딩 구매 추천 이벤트의 경우, 추천받은 사람이 당첨 발표 시점에 슈퍼리딩 정회원이 아닌 경우, 혜택적용에서 제외됩니다.
  • 형제, 가족간 추천 등 부적절한 방법으로 이벤트에 참여했다고 자사가 판단하는 경우, 별도의 고지 없이 해당 회원의 이벤트 당첨이 제외될 수 있습니다.
  • 네이버페이 상품권, 신세계 상품권은 회원정보에 기재된 회원 휴대폰 번호로 발송되며, 잘못된 정보 기재로 인한 경품 오발송 건에 대해서는 재발송 처리되지 않습니다.
  • 네이버페이 상품권, 신세계 상품권은 24년 4월 17일(수) LMS를 통해 개별 발송됩니다.
  • 경품은 5만원 초과 시 제세공과금(경품가의 22%)이 발생되며, 당첨자 본인 부담입니다. 제세공과금 입금, 개인정보 수집 및 이용 동의서 전달을 위해 개별 연락 드립니다.
  • 내 추천 링크를 통해 친구가 슈퍼리딩이 아닌 슈퍼브이 스터디 무료체험/구매 했을 경우, 슈퍼브이 스터디 친구추천 경품이 제공됩니다. (상기 슈퍼리딩 혜택과 다름)
